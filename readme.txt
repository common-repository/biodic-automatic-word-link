Biodic Automatic Word Link (AWL) crea enlaces automáticos al Diccionario de Biología BioDic. Todas las palabras que se encuentren en tu texto serán automáticamente enlazadas para que tus usuarios puedan aprender con tus entradas..


== Description ==

BioDic Automatic Word Link es un plugin para WordPress que habilita la posibilidad de buscar en tus entradas y añadir de forma automática enlaces del Diccionario de Biología BioDic, así tus lectores podrán aprender a la vez que leen tus entradas. El plugin cuenta con la opción de mostrar un tooltip para que el usuario pueda leer rápidamente la definición completa o parte de ella, directamente en tu web.

Es importante conocer que este plugin puede introducir muchos enlaces en tus entradas, y ralentizar un poco la carga de la misma, se recomienda un plugin de caché para ello.


== Installation ==

1. Sube el contenido del zip a la carpeta `/wp-content/plugins/` 
2. Activa el plugin a través del menú 'Plugins' del panel de WordPress
3. Configura las opciones de visualización en la sección 'BioDic AWL' del panel de control de WordPress
